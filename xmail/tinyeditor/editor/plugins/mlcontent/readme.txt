 MLcontent plugin for tinyeditor
 by ralf57
 12-03-2005
###############################

This plugin allows you and your site's users
to add multi-language content into tinyeditor's area.
You MUST have "THE EASIEST MULTILANGUAGE HACK" by GIJOE
applied to your xoops installation before enabling and using this plugin.
The hack can be found at http://www.peak.ne.jp/xoops/

Installation instructions:
  * Copy the "mlcontent" directory to the plugins directory of tinyeditor (/xoops/modules/tinyeditor/editor/plugins).
  * Enable the plugin from tinyeditor's config options
  * Add the button to tinyeditor toolbar using the toolbar configurator

That's all!
